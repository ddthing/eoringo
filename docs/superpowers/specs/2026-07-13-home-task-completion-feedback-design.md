# 오늘 체크 완료 피드백 설계

## 목표

오늘 체크에서 숙제를 완료했을 때 항목이 즉시 사라지는 대신, 완료 결과를 충분히 인지한 뒤 자연스럽게 목록에서 정리되도록 한다. 숙제 데이터는 애니메이션과 관계없이 오늘 체크, 숙제 탭, 숙제 진행률에 즉시 연동되어야 한다.

## 적용 범위

- 완료 후 퇴장 애니메이션은 Home의 오늘 체크 목록에만 적용한다.
- 단일 체크 숙제와 목표 횟수가 있는 숙제를 모두 지원한다.
- 숙제 탭의 목록 표시 방식은 변경하지 않는다.
- Zustand 숙제 저장 구조와 완료 기록 형식은 변경하지 않는다.

## 상호작용

### 단일 체크 숙제

1. 사용자가 미완료 항목을 누르면 전역 완료 상태를 즉시 저장한다.
2. 해당 행은 초록색 체크, 옅은 민트색 배경, `완료` 배지를 표시한다.
3. 완료 상태를 900ms 동안 유지한다.
4. 220ms 동안 오른쪽으로 미세하게 이동하며 투명해지고 높이가 접힌다.
5. 퇴장이 끝나면 행을 오늘 체크 목록에서 제거한다.

### 횟수형 숙제

- 목표 횟수 미만에서는 기존처럼 횟수만 즉시 변경한다.
- 마지막 증가로 목표 횟수에 도달하면 단일 체크 숙제와 같은 완료 효과를 적용한다.
- 감소 동작과 목표 도달 전의 증감에는 퇴장 효과를 적용하지 않는다.

## 데이터 흐름

- 전역 숙제 저장은 사용자 입력 시점에 즉시 갱신한다.
- 오늘 체크, 숙제 탭, Home 숙제 진행률은 기존의 동일한 Zustand 데이터를 사용하므로 즉시 함께 갱신된다.
- Home은 방금 완료된 항목의 표시 정보만 로컬 퇴장 상태로 잠시 보존한다.
- 로컬 퇴장 상태는 저장 데이터의 완료 여부를 대체하거나 지연하지 않는다.
- 화면 이동, 컴포넌트 해제 또는 캐릭터 전환 시 임시 퇴장 상태와 타이머를 정리한다. 이미 저장된 완료 기록은 유지한다.

## 컴포넌트 책임

### `HomeTodayCheck`

- 퇴장 중인 항목과 타이머를 관리한다.
- 입력 전에 완료될 항목의 표시 정보와 목록 위치를 보존한다.
- 저장 동작을 즉시 호출하고, 완료된 행을 기존 위치에 잠시 렌더링한다.
- 퇴장 완료 후 로컬 표시 상태만 제거한다.

### `HomeTodayCheckItem`

- 일반 상태와 완료 피드백 상태를 시각적으로 렌더링한다.
- 완료 피드백 중에는 중복 입력을 막는다.
- 체크, 배경, 배지 및 퇴장 단계에 필요한 접근성 상태와 CSS 클래스를 제공한다.

### 도메인 및 Store

- 기존 `getHomeTodayTaskGroups`의 완료 계산과 `useTaskStore`의 저장 로직을 유지한다.
- 애니메이션 상태를 전역 Store나 영속 데이터에 추가하지 않는다.

## 연속 입력과 정합성

- 서로 다른 여러 항목을 빠르게 완료하면 각 항목을 독립적으로 퇴장시킨다.
- 이미 퇴장 중인 항목의 버튼과 횟수 조작은 비활성화하여 중복 완료를 방지한다.
- 진행률과 그룹 완료 숫자는 저장 직후 갱신한다. 퇴장 중인 행이 잠시 보이더라도 숫자는 최신 완료 상태를 보여준다.
- 퇴장 중 마지막 항목 때문에 그룹 상태가 `complete`가 되면 완료 행이 사라진 다음 그룹 완료 메시지를 표시한다.

## 모션과 접근성

- 새 애니메이션 라이브러리를 추가하지 않고 React 로컬 상태와 CSS 전환만 사용한다.
- 완료 배지는 시각적 피드백이며 기존 버튼의 접근 가능한 이름을 불필요하게 바꾸지 않는다.
- `prefers-reduced-motion: reduce`에서는 이동과 높이 축소를 생략한다. 완료 상태는 900ms 동안 표시한 뒤 즉시 목록에서 제거한다.
- 타이머는 컴포넌트 해제 시 모두 해제해 해제된 컴포넌트의 상태를 갱신하지 않는다.

## 성능

- 전역 애니메이션 큐를 만들지 않고 현재 보이는 완료 행만 로컬 상태에 보관한다.
- 각 완료 항목당 타이머 하나와 작은 표시 스냅샷 하나만 사용한다.
- 전체 숙제 목록이나 Store 구조를 복제하지 않는다.

## 검증

- 단일 체크 숙제가 즉시 저장되면서 완료 피드백 후 제거되는지 확인한다.
- 횟수형 숙제가 마지막 증가에서만 완료 피드백을 표시하는지 확인한다.
- 오늘 체크에서 완료한 결과가 숙제 탭과 진행률에 즉시 반영되는지 확인한다.
- 연속으로 여러 항목을 완료해도 각각 한 번만 저장되고 자연스럽게 제거되는지 확인한다.
- 캐릭터 전환과 화면 이동 시 타이머 오류나 데이터 유실이 없는지 확인한다.
- 모션 감소 설정에서 이동 없이 완료 상태가 정리되는지 확인한다.
- 390px 화면에서 배지와 제목이 충돌하거나 가로 스크롤이 생기지 않는지 확인한다.
- 전체 테스트와 프로덕션 빌드를 실행한다.
